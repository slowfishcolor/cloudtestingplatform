云检测平台

云检测平台是云制造理论在测试行业的落地应用，平台 1.0 版本针对电子组装制造业的模块及整机测试环节的测试模式进行了研发适配，实现了实验室环境下对典型测试仪器的接入及虚拟化服务化封装，可以通过 Web 应用的方式来调用仪器仪表，执行测试任务。
